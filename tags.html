Tags HTML - Desafio

HTML é uma linguagem de marcação de hipertexto, e tem como principal função marcar e definir a estrutura de uma página na WEB.

Vamos ver algumas das tags mais importantes, que foram estudadas no curso de Formação HTML Web Development, e que o HTML utiliza na sua estrutura.

O que são tags HTML?

As tags são usadas para informar ao navegado a estrutura do site. Ao se utilizar de tags HTML, elas serão interpretados pelo navegador, e são renderizadas na tela, produzindo o conteúdo visual da página que nós vemos, com cores, fonts, imagens etc ...

As tag devem estar dentro dos sinais de chevron (sinal de "maior que" e "menor que")

Atributos

Os atributos são usados para personalizar as tags, modificando sua estrutura e funcionalidade. Atributos são utilizados para atribuir uma classe ou id a um elemento

Conheça aqui, todas as tags que foram explicadas nas aulas: <h1> até <h6>, <p>, <mark>, <small>, <i>, <u>, <strong>, <ol>, <ul>, <li>, <a>, <hr>, <sub>, <sup>, <bloquequote>, ;

Utilizar novas tags sugeridas: <font>, <del>, <abbr> (a ideia é buscar estas tags na internet, entender como ela funciona e utilizar no texto).

TAG <p>

A tag <p> cria um bloco compreendido como um parágrafo pelos navegadores, idealmente para agrupar textos.

O elemento HTML < p> (ou Elemento HTML Parágrafo) representa um parágrafo do texto. Parágrafos são, geralmente, representados em mídia visual , como bloco de texto que são separados dos blocos adjacente por espaços brancos verticais e/ou recuo de primeira-linha. Parágrafos são elementos em bloco . Developer Mozilla

TAG <mark>

No HTML 5 o novo elemento mark tem como função destacar, realçar um texto, como por exemplo, quando fazemos uma busca por uma palavra em um mecanismo de busca e este retorna os resultados encontrados destacados com uma formatação diferente (highlighted text)

Seu uso mais efetivo é combinado com um estilo CSS que lhe adiciona uma cor de fundo.

Sintaxe:

<mark> Contents ... </mark>
  1. O violão, violino o cavaquinhosão apenas alguns dos mais conhecidos instrumentos musicais de corda ...
  2. O Trompete, Trombone, Tuba,Clarinete, Flauta, Saxofone, são instrumentos de sopro ...

TAG <strong>

A tag <strong> renderiza o texto contido nela com um reforço gráfico (negrito), e informa a mecanismos de busca e outras plataformas que aquele conteúdo tem maior significação do que os demais conteúdos apresentados no documento. A tag <strong> é uma tag semântica, e atua tanto visualmente quanto em nível de meta-informação para outros mecanismos da web. Os resultados visuais são idênticos ao da tag <b>, mas esta não produz nenhuma meta-informação, ficando restrita a formatação de texto.

Sintaxe:

<strong>, Conteúdo... </strong>

Exemplo:

Sou o destaque desse trecho, por isso sou muito importante, pois eu sou o <strong> e não apenas o negrito <b>;

TAG <ol>

Listas ordenadas ou numerada

O marcador <ol> </ol>"Ordered List Element" ou lista ordenada, por si só é um conjunto vazio, define a abertura e fechamento de uma lista.

Deve conter no mínimo uma tag <li>, </li> dentro do conjunto.

Dentro de uma lista ordenada degraus podem ser colocados se forem necessários, os degraus podem ser uma lista ordenada ou não, depende da necessidade.

Ordenação em numéricos arábicos:

A ordenação padrão é com números arábicos. Esta condição, se desejada, não precisa ser declarada.

TAG <ul>

Listas não Ordenadas

As listas não numeradas são usadas para listar itens, sem se preocupar com sua sequência. Chamamos de lista de marcadores apenas.

As tags usadas para criar uma lista não ordenada são <ul> e <li>. A tag <ul> é usada para definir a lista e a tag <li> é usada para cada item da lista.

TAG <li>

A tag <li> em HTML é usada para definir o item da lista em um documento HTML. É usado em uma Lista ordenada <ol> ou Lista não ordenada <ul>.

Observação: a tag final pode ser omitida se o item da lista for seguido imediatamente por outro elemento <li> ou se não houver mais conteúdo em seu elemento pai.

Sintaxe:

<li>, item </li>

TAG <a>

A tag <a> em HTML (conhecida como tag âncora), com o atributo href cria-se um hiperligação nas páginas web, arquivos, endereços de emails, ligações na mesma página ou endereços na URL. O conteúdo dentro de cada <a> precisará indicar o destino do link.

A tag <a> , é responsável por vincular os demais documentos que o HTML possui, links internos e externos.

Atributos

Além de href, a tag <a> possui alguns outros atributos, e cada um deles tem o seu papel específico. Esses atributos são: target e rel.

O href (hypertext reference),referência para hipertexto, que nada mais é do que a tag onde colocamos a URL do site que queremos vincular ao nosso HTML. É necessário adicionarmos a URL por completo, isso inclui o protocolo https. Ao incluirmos a URL do site desejado, basta colocarmos uma palavra ou frase entre o fechamento e abertura da tag <a>. Quando clicarmos nessa palavra/frase, seremos direcionados ao site vinculado. É possível também utilizarmos a tag <a> apenas para voltar ao topo da nossa página, para isso, inserimos uma cerquilha.

Sintaxe:

<a href="#">Topo </a>
<a href="https://www.dio.me.com/">Dio.me</a>

target

o atributo target, é utilizado para especificar onde iremos visualizar o link após clicarmos, seja ele interno ou externo.

Para que o link seja aberto em uma nova aba:

<a href="https://www.dio.me.com/" target="_blank">Dio.me</a>

Para que o link seja aberto na mesma aba, sendo esse seu comportamento padrão, basta declarar:

<a href="https://www.dio.me.com/" target="_self">Dio.me</a>

rel

o atributo rel (relationship), relação. É muito importante que saibamos que o atributo rel contém alguns valores, sendorel eles distintos uns dos outros. Iremos abordar os valores de nofollow, noopener e noreferrer, que está ligado diretamente com o noopener.

<a href="https://www.dio.me.com/" rel="nofollow">Dio.me</a>

Estamos informando para o Google não dar importância para o link que estamos referenciando, ou seja, ele irá ler o nosso link de âncora, apresentar ao usuário, porém, o mesmo não será ranqueado.

Sua principal finalidade é evitar com que pessoas paguem desenvolvedores de sites para aumentarem a visualização dos seus links (ranqueamento). Isso é feito quando declaramos a tag de âncora sem o atributo rel com o valor de nofollow.

TAG <hr>

A tag <hr> "Horizontal Rule" é um elemento HTML que representa uma quebra de linha temática entre elementos HTML, é exibido como uma linha horizontal separando os conteúdos de uma página web.

Sintaxe:

<hr>
Atenção: A tag <hr> é considerada uma tag vazia, pois não requer tag de fechamento.

Importante: A tag <hr> está em todas as sessões desse documento.

Propriedades

  • size= Medida padrão em unidade pixel que define a altura da linha. A unidade de medida "px" não necessita ser inserida.
  • Align= Define o alinhamento em relação à pagina. O Alignhamento pode ser à esquerda, no centro, ou à direita. Sendo a alinhamento padrão, caso não seja definido, o centro
  • width= Largura da linha horizontal que é determinado em porcentagem do local onde a tag <hr>, está inserida ou em pixels.
  • noshade:Quando está propriedade está definida a linha será sombreada. Se a propriedade não for definida e não houver a inclusão de cor alternativa a padrão a linha será sombreada
  • color=Define a cor da linha horizontal. A cor padraão da tag é #A0A0A0. Se nenhuma propriedade "color" não for definida a linha assumira essa cor.

Exemplos:

Uma tag <hr> sem nenhuma propriedade, será apresentada assim no navegador:
<hr>

Uma tag <hr> com a propriedade "size=10", será apresentada assim no navegador:
<hr size=10/>

Uma tag <hr> com as propriedades "align="left",width="30%" e "size=5" assim aparecerá no navegador:
<hr align="left" width="30%" size="5"/>

Uma tag <hr> com a propriedade "noshade" ativada, e as propriedades width="30%" size="5" assim aparecerá no navegador:
<hr width="30%" size="5" noshade />

Uma tag <hr> com as propriedades width="30%" size="5" color="Yellow" assim aparecerá no navegador:
<hr width="30%" size="5" color="blue" />



TAG <sub>

O marcador <sub> </sub> "Subscript Element" ou subscrito é utilizado para escrever um texto com uma fonte menor e com o alinhamento abaixo da linha normal.

Texto subscrito pode ser utilizado para as fórmulas químicas, tal como H2O

Sintaxe:

<sub>...</sub>

Exemplo:

O benzeno é um hidrocarboneto aromático cuja fórmula química é:
C6H6 - Seis átomos de carbono e seis de hidrogênio.

TAG <sup>

O <sup>tag define texto sobrescrito. texto sobrescrito parece metade de um caráter acima da linha normal, e às vezes é processado em uma fonte menor. Superscript texto pode ser utilizado para notas de rodapé, como WWW [1] .

Dica:

Use o <sup> tag para definir o texto subscrito.

Sintaxe:

<sup>*</sup>

TAG <blockquote>

A tag <blockquote> é usada para exibir as citações longas (seção que é citado de outra fonte).

Os navegadores normalmente renderizam o texto encontrado na tab <blockquote> como um texto recuado.

Atributo

Contém um único atributo cite que é usado para especificar a fonte de citação

Dicas

Use <q> para in-line cotações (curtas).

Chevron (tipografia)

Chevron é um sinal de pontuação duplo que se coloca em um fragmento de texto, devendo ser traduzido como "divisa". Matemáticamente também significa "menor que" e "maior que".

Na Informática, em XML e HTML os chevrons são utilizados para a concepção de tags para aplicar formatação de conteúdo especifico que circunda.

TAG <del>

A tag <del> define o texto que foi excluído de um documento. Os navegadores geralmente riscam uma linha através do texto excluído.

Os navegadores tradicionalmente renderizam o texto encontrado na tag <del>como texto tachado. Além disso Você pode usar a tag complementar chamada tag <ins>para marcar o novo texto que foi inserido em um documento.

TAG <abbr>

A tag <abbr> </abbr> "Abbreviation Element" abreviatura, define uma abreviação ou acrônico é uma tag utilizada para se inserir uma sigla em uma página HTML.

É útil para fornecer informações aos navegadores, leitores de telas, buscadores, sites de tradução como também verificadores de ortografia.

O marcador <abbr> </abbr> pode conter a característica "title=", opcional, onde se coloca o nome completo do que aquela sigla representa. Este atributo fornece principalmente informações aos buscadores.
Ex: <abbr title="Cascading Style Sheets">CSS</abbr>
<abbr title="Linguagem de marcação do Hypertext">HTML</abbr>

Sintaxe:

<abbr>...</abbr>

Dica:

Use o atributo global para mostrar a descrição da abreviatura/acrônimo ao passar o mouse sobre o elemento

Exemplo:

Autoridade monétaria do brasil, BCB

Organização Mundial da Saúde WHO Fundada em 1948
